README — Proyecto Web HTML y CSS
Descripción del proyecto

Este proyecto consiste en el desarrollo de un sitio web realizado en HTML5 y CSS3 para la materia de Informática General de la carrera de Artes Multimediales.

El sitio fue construido utilizando estructuras semánticas y estilos trabajados en clase, incorporando:

Maquetación con contenedores
Menú horizontal
Estilos tipográficos
Galerías de imágenes
Inserción de video de YouTube
Uso de etiquetas semánticas
Fondos personalizados
Organización visual mediante clases e IDs
Diseño adaptable básico
Estructura del sitio

El proyecto está compuesto por distintas páginas HTML conectadas entre sí mediante un menú de navegación.

Páginas realizadas
index.html
biografia.html
discografia.html
show.html
